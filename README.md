# Usage
<!-- usage -->
```sh-session
$ npm install -g web-stalker
$ web-stalker COMMAND
running command...
$ web-stalker (-v|--version|version)
web-stalker/1.0.0 win32-x64 node-v10.16.3
$ web-stalker --help [COMMAND]
USAGE
  $ web-stalker COMMAND
...
```

## `web-stalker hello [FILE]`

describe the command here

```
USAGE
  $ web-stalker hello [FILE]

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ web-stalker hello
  hello world from ./src/hello.ts!
```

_See code: [src\commands\hello.ts](https://github.com/fivetwentysix/web-stalker/blob/v1.0.0/src\commands\hello.ts)_

## `web-stalker help [COMMAND]`

display help for web-stalker

```
USAGE
  $ web-stalker help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v2.2.3/src\commands\help.ts)_
